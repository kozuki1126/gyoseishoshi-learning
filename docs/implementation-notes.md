## 技術仕様

### フロントエンド
- **React 18** (Next.js 14.1ベース)
- **ReactMarkdown** (リアルタイムプレビュー)
- **Tailwind CSS** (UIスタイリング)
- 動的インポートによるSSRサポート

### バックエンド
- **Node.js** (Next.js APIルート)
- **formidable** (ファイルアップロード処理)
- **File System API** (ローカル保存)
- 50MBまでのファイルサイズ制限

## 使用方法

1. 依存関係のインストール:
```bash
npm install
```

2. 開発サーバー起動:
```bash
npm run dev
```

3. 管理画面へのアクセス:
```
http://localhost:3000/admin/content
```
